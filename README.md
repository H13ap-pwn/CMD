- `systeminfo` : Thông tin hệ điều hành
- `ipconfig` : Thông tin mạng
  + `ipconfig /all` : Thông tin mạng đầy đủ
- `ping <...>` : Xem kết nối được đến ... ko, và tốc độ truyền tin
- `tracert <...>` : Vẽ lại con đường dữ liệu đi qua, xem lỗi cụ thể ở đâu và cho biết các IP/Router trung gian
- `nslookup <...>` : Tra cứu một máy chủ hoặc tên miền và trả về IP của nó
- `netstat` : Hiển thị các kết nối mạng và các cổng lắng nghe
  + `netstat -abon` : Đầy đủ hơn
